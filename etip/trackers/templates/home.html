Welcome to Exodus Tracker Investigation Platform

What is ETIP?

ETIP is meant to ease investigations on tracker detection for the platform εxodus.

It offers the following features:

  • track all modifications on trackers
  • detect rules collisions for both network and code signature

How to contribute?

If you wish to help us identify new trackers, you can easily request an ETIP account.

Once you have an account, you can:

Create new trackers profiles

Create new tracker

Extend the content of existing trackers profiles

See all trackers

Review trackers before they are sent to εxodus

Review trackers

Curious about ETIP?

ETIP is free and open source, feel free to check its GitHub page.
